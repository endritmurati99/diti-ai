# Habits & Weekly Goals DB — Schema

## Zweck
Tracking von Gewohnheiten und Wochenzielen. SoR fuer Habit-Metriken.

## Properties

| Property | Typ | Werte | Beschreibung |
|----------|-----|-------|-------------|
| Name | Title | — | Ziel/Gewohnheit |
| Week | Rich Text | KW-YYYY-WW | Kalenderwoche |
| Category | Select | Health, Productivity, Learning, Personal | Kategorie |
| Target | Rich Text | — | Zielwert (z.B. "4x Training") |
| Actual | Rich Text | — | Ist-Wert |
| Status | Select | On Track, Behind, Achieved, Missed | Status |
| Notes | Rich Text | — | Anmerkungen |
