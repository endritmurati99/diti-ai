# Health Weekly DB — Schema

Woechentliche Gesundheits-Aggregate. Keine Rohdaten, nur Summaries.

| Steps Avg | Number | — | Durchschnittliche Schritte/Tag |
| Sleep Avg | Number | — | Durchschnittlicher Schlaf (Stunden) |
| Training Count | Number | — | Anzahl Trainingseinheiten |
| Weight | Number | — | Gewicht (kg) |

| Notes | Rich Text | — | Anmerkungen |
